# codegloss · Next.js + @next/mdx

This page is rendered from an MDX file. The fenced sandbox below is rewritten
into a `<CodeGloss />` component by `remark-codegloss`, which the import map in
`next.config.mjs` registers as a remark plugin.

```js sandbox fib.js
function fib(n) {
  return n < 2 ? n : fib(n - 1) + fib(n - 2);
}
```
