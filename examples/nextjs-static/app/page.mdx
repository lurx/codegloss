# codegloss · Next.js static export

The same MDX setup as `nextjs-app`, but with `output: 'export'` so the build
produces a static `out/` directory you can host on any CDN.

```js codegloss fib.js
function fib(n) {
  return n < 2 ? n : fib(n - 1) + fib(n - 2);
}
```

